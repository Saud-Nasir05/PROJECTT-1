## 📌 Overview
BizGenie is a comprehensive full-stack platform designed to help entrepreneurs evaluate, plan, and execute business ideas. It provides dynamic budget allocations, growth scaling timelines, and detailed execution roadmaps tailored to specific local markets.

## ✨ Key Features
* **Dynamic Financial Projections:** Get automated CapEx, OpEx, and ROI calculations based on business type.
* **Execution Roadmaps:** Step-by-step phased timelines for launching a business.
* **Local Market Strategy:** Competitor analysis and demographic breakdowns.
* **Interactive UI:** Built with React and Tailwind CSS / DaisyUI for a seamless experience.

## 🛠️ Tech Stack
* **Frontend:** React.js, Tailwind CSS, DaisyUI, React Router
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Mongoose
* **Deployment:** Vercel
